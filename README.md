# Before Us Universe

> **Digital Memory Archives – LIVE CHAT + MESSAGE BOARD**  
> A retro-futuristic marketing portal for the *Before Us* book universe.

## Live Demo
https://robustpvp-lang.github.io/before-us-universe/

## Features
- Glitchy CRT boot sequence
- Yahoo-Messenger-style **live chat**
- **Message board** (last 10 posts, `localStorage`)
- **Yaga AI** (Cortana × Baba Yaga) moderates + answers lore
- 100% static **or** real-time with Node server

## Local Development
```bash
# 1. Static (GitHub Pages)
git clone https://github.com/robustpvp-lang/before-us-universe.git
open index.html

# 2. Real-time server
cd server
npm install
npm start
# → open http://localhost:3000
